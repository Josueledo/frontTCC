Frequently asked questions

Find answers to common questions about system security, reports, and account management.

General Report

It provides an overview of your system’s security status, including detected threats, blocked IPs, and user activity summaries.

How often are reports updated?
+
Reports are updated in real time as new data is collected and analyzed.
Can I export security reports?
+
Yes. In the “General Report” section, click Export Report to download a summary in CSV or PDF format.
Can I customize the report filters?
+
Yes. You can filter by date, IP address, or event type to view specific information.
Blocked IPs

IPs are automatically blocked when the system detects suspicious or unauthorized access attempts.

How can I unblock an IP?
+
Go to the “Blocked IPs” tab, find the IP you want to unblock, and click Unblock.
What happens after I unblock an IP?
+
Once unblocked, the IP regains system access, but monitoring continues to detect future threats.
Can I export the list of blocked IPs?
+
Yes. You can export the current list for auditing or investigation purposes.
